## Catatan Developer

- Tidak ada iterasi dalam perhitungan As — solusi analitik eksak digunakan
- `ρ_max = 0.75 × ρ_balanced` memastikan zone terkontrol tarik sehingga φ = 0.90 valid
- Verifikasi εt ≥ 0.005 tidak dilakukan eksplisit; dijamin via batas ρ_max
- Tulangan rangkap (doubly reinforced) belum diimplementasikan
