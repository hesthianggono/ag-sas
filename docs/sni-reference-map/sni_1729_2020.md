### Tabel B4.1b — Klasifikasi Kelangsingan Elemen Tekan untuk Profil I

**Modul terkait:** `app/calculation/steel/beam_calculator.py`

#### Sayap profil I simetris ganda dalam lentur

| Parameter | Simbol | Rumus | Satuan |
|-----------|--------|-------|--------|
| Rasio kelangsingan sayap | λ_f | (b/2) / tf | - |
| Batas kompak | λ_pf | 0.38 × √(E/Fy) | - |
| Batas tidak kompak | λ_rf | 1.0 × √(E/Fy) | - |

| Kondisi | Klasifikasi |
|---------|------------|
| λ_f ≤ λ_pf | Kompak (compact) |
| λ_pf < λ_f ≤ λ_rf | Tidak kompak (noncompact) |
| λ_f > λ_rf | Langsing (slender) — belum diimplementasikan |

### Pasal F2.1 — Momen Plastis dan Mn untuk Penampang Kompak + Lb ≤ Lp

**Modul terkait:** `app/calculation/steel/beam_calculator.py`

| Parameter | Simbol | Rumus | Satuan |
|-----------|--------|-------|--------|
| Momen plastis | Mp | Fy × Zx | N·mm → kN·m |
| Momen nominal (kompak) | Mn | Mp | kN·m |

Berlaku jika: (1) sayap kompak, dan (2) Lb ≤ Lp (tidak ada LTB)

---

### Persamaan F3-1 — Mn untuk Penampang Tidak Kompak (Flange Local Buckling)

**Modul terkait:** `app/calculation/steel/beam_calculator.py`

```
Mn = Mp − (Mp − 0.7 × Fy × Sx) × (λ_f − λ_pf) / (λ_rf − λ_pf)
```

| Parameter | Keterangan |
|-----------|------------|
| Mp | Momen plastis = Fy × Zx |
| 0.7 × Fy × Sx | Momen batas bawah (inelastic local buckling) |
| (λ_f − λ_pf)/(λ_rf − λ_pf) | Faktor interpolasi linier |

## Catatan Developer

- LTB (Lateral-Torsional Buckling) tidak dicek — Lb ≤ Lp diasumsikan. Wajib diverifikasi per proyek.
- Modul tidak menangani penampang slender (λ_f > λ_rf)
- Database profil baja tersedia di `app/models/steel_profile.py` dengan 34 profil (WF, H-Beam, CNP, UNP)
- Penambahan profil baru via `app/services/steel_profile_service.py`
